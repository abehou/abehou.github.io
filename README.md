## Setup

### Local Development

1. Simply open `index.html` in your browser
2. No build process or dependencies required!

### Hosting on GitHub Pages

1. Create a new repository on GitHub named `username.github.io` (replace `username` with your actual GitHub username)

2. Push your code to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/username/username.github.io.git
   git push -u origin main
   ```

3. Enable GitHub Pages:
   - Go to your repository settings
   - Navigate to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click "Save"

4. Your site will be live at `https://username.github.io` in a few minutes!

## Customization

### Update Your Information

1. **index.html**: Replace placeholder text with your name, position, interests, and contact info
2. **publications.html**: Add your papers, conference presentations, etc.
3. **experiences.html**: Add your work experience and education
4. **blog.html**: Add your blog posts (or link to external blog)

### Customize Styling

Edit `style.css` to change:
- Colors
- Fonts
- Spacing
- Layout

### Add Custom Domain (Optional)

1. Create a file named `CNAME` in the root directory
2. Add your custom domain (e.g., `www.yourdomain.com`)
3. Configure DNS settings with your domain provider
